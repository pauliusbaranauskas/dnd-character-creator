# D&D Character Creator

A Python application built with PyQt6 for creating and managing D&D characters.

## Features
- Create characters by selecting class, race, and stats.
- Save and load characters to/from files.
- Level up characters with stat increases and new skills.
- Review character history and undo updates.

## Setup Instructions

### 1. Create a Virtual Environment
It is recommended to use a virtual environment to manage dependencies and avoid conflicts.

**On macOS and Linux:**
```bash
python3 -m venv venv
```

### 2. Activate the Virtual Environment
**On macOS and Linux:**
```bash
source venv/bin/activate
```

### 3. Install Dependencies
Once the virtual environment is activated, install the required packages:
```bash
pip install -r requirements.txt
```

### 4. Run the Application
```bash
python3 app.py
```
